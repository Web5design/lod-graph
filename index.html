Linked Open Data Graph

This is a Protovis version of the (much more useful) Linked Open Data Cloud. It is based on the same underlying data, by using the Comprehensive Knowledge Archive Network (CKAN) API to pull all the packages in the the lodcloud group. If your screen goes crazy cross your fingers and try to reload the page a couple times. The colors reflect the CKAN rating: 1, 2, 3, 4, 5, unrated datasets are white. Nodes with more than 10,000,000 triples have short names next to them. You should be able to hover over any node to get its full name, and if you click on it you will be taken to the package details page at CKAN. More about how this graph was generated can be found on Github. Comments/questions welcome at ehs@pobox.com.
